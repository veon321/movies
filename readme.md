# Movies App

A sleek, modern, and fully interactive frontend Movie Search application that lets you search for movies, filter by year, and save favorites. Built with vanilla technologies, it features a responsive dynamic grid layout and a sophisticated dark UI.

## Live Demo

https://veon321.github.io/movies/

## Features

- Search movies by title using the OMDb API
- Filter results by year
- Add or remove favorites
- Favorites persist via localStorage
- Toggle between All and Favorites views

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OMDb API
